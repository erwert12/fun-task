# Fun Task (Экзамен для интернов (нас))


CLI-приложение на PHP 7.4 с использованием:
                            ![gif](https://99px.ru/sstorage/1/2011/01/image_1280111141904933553.gif)
- Symfony Console
- PHPStan
- MyCLabs Enum

Реализовано как консольное приложение.

---

## Требования

- PHP 7.4
- Composer

---

## Установка

Установить зависимости:

```bash
composer install
```

---

## Запуск CLI

Запуск приложения:

```bash
php bin/console
```

Список доступных команд:

```bash
php bin/console list
```

---

## Запуск PHPStan 

```bash
composer stan #Собственноручно написанный скрипт
```
